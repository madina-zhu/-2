## ✅ Реализованные компоненты

### 1. Класс Tensor с Autograd (`tensor.py`)
- Хранение данных и градиентов
- Построение графа вычислений
- Ручное обратное распространение

### 2. Слои и модели (`layers.py`)
- `Linear` - полносвязный слой с инициализацией Xavier
- `Sequential` - контейнер для последовательного применения слоев

### 3. Функции активации (`activations.py`)
- `ReLU` - выпрямленная линейная активация
- `Sigmoid` - сигмоидальная активация
- `Softmax` - для многоклассовой классификации

### 4. Функции потерь (`losses.py`)
- `MSELoss` - среднеквадратичная ошибка (регрессия)
- `CrossEntropyLoss` - кросс-энтропия (классификация)

### 5. Оптимизаторы (`optimizers.py`)
- `SGD` - стохастический градиентный спуск
- `Momentum` - SGD с моментом
- `Adam` - адаптивный оптимизатор
- `GradientClipping` - обрезка градиентов (обертка)
